Command
v1.2.0
Bootstrap a multiple sequence alignment.
cytos msa bootstrap [OPTIONS] <INPUT> <OUTPUT>
Parameters
Options
--replicates
v1.2.0
Create this many bootstrap replicates.
Default value: 1
--seed
v1.2.0
Random number generation seed (e.g. 42)
Positional arguments
<INPUT>
v1.2.0
Read the FASTA file from this path.
<OUTPUT>
v1.2.0
Write the bootstrapped FASTA files to this directory.
Example
The following example one FASTA file input.fa, then sub-samples each alignment (with replacement) to the directory /tmp/replicates.
cytos msa bootstrap --replicates 1 input.fa /tmp/replicates

>baz
